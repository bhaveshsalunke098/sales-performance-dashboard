# 📊 CodeAlpha Data Analytics Internship – Task 3

## Sales Performance Dashboard

### 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this task is to create an interactive dashboard that transforms raw sales data into meaningful visual insights, enabling effective business analysis and data-driven decision-making.

---

## 🎯 Objectives

- Monitor overall sales performance
- Analyze revenue trends
- Identify top-performing products
- Visualize key business metrics
- Generate actionable business insights

---

## 🛠️ Technologies Used

- Power BI
- CSV Dataset
- Data Visualization
- Business Intelligence

---

## 📂 Files Included

- `Sales Performance Dashboard.pbix` – Power BI Dashboard
- `sales_data.csv` – Dataset used for analysis
- `README.md` – Project Documentation

---

## 📊 Dashboard Features

- Sales Overview
- Revenue Analysis
- Product Performance Tracking
- Category-wise Analysis
- Interactive Filters and Slicers
- Trend Analysis

---

## 📈 Key Insights

- Identified top-performing products and categories.
- Analyzed sales trends and revenue growth.
- Evaluated overall business performance.
- Created interactive visualizations for effective reporting.
